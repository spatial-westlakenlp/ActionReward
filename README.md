## 🚀 Released Works

### VeMo: Zero-Shot Text-to-Motion Evaluation using Video Language Models

[Docs](VeMo/README.md) · [Paper](VeMo/assets/paper.pdf) 

**Takeaways**

- Evaluates whether generated text-to-motion outputs match their prompts.
- Renders 3D motions into videos and scores alignment with pretrained video-language models.
- Includes human-annotated benchmark resources for meta-evaluating T2M metrics.
